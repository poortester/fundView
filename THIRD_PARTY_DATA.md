# 第三方数据与接口说明

本项目会调用若干第三方服务。当前定位是本地个人使用和工程实验，不建议直接作为公开商业服务使用。

## 东方财富 / 天天基金

用途：

- 市场指数行情
- 基金净值和估值
- 基金持仓穿透
- 基金公告
- 基金相关正式资讯

注意事项：

- 当前实现调用的是东方财富/天天基金网页公开资源，并非本项目持有的官方 SDK 授权。
- 这些接口可能变化、限流、下线或返回不完整数据。
- 若用于商业产品、公开服务或高频访问，请自行确认服务条款，并替换为授权数据源。

## 百度 OCR

用途：

- 可选的持仓截图识别。

需要用户自行配置：

- `BAIDU_OCR_API_KEY`
- `BAIDU_OCR_SECRET_KEY`

注意事项：

- 持仓截图可能包含个人金融信息。
- 使用前请确认你接受百度 OCR 的服务条款和隐私政策。

## 大模型服务

用途：

- 可选的研究员分析增强。

支持形式：

- OpenAI 兼容 Chat Completions 接口
- Anthropic Messages 接口

注意事项：

- Prompt 可能包含基金名称、行情指标、持仓穿透、公告摘要和研究上下文。
- 不建议把敏感账户数据发送给不可信模型服务。
- 页面中配置的模型密钥会保存到本地 `llm-settings.local.json`，该文件已加入 `.gitignore`。
