# 安全说明

## 使用定位

本项目默认面向本地个人使用，不是已经完成安全加固的公开多用户 SaaS。

## 不应提交的内容

请不要提交：

- `.env.local`
- Turso/libSQL token
- 百度 OCR 密钥
- 大模型 API Key
- `llm-settings.local.json`
- SQLite / local.db 数据库文件
- 持仓截图、账户截图或个人导出数据

如果密钥已经提交或在公开渠道出现，请立即到对应服务商后台旋转密钥。仅删除 Git 文件或重新提交无法保证安全。

## 对外部署前建议

如果要部署到公网，请至少补充：

- 登录认证
- 用户权限隔离
- 请求限流
- CORS 与代理规则审查
- 日志脱敏
- 文件上传限制
- OCR / LLM 数据发送确认
- 第三方数据源条款审查

## 漏洞反馈

如果你发现安全问题，请优先通过私有渠道联系维护者。不要在公开 issue 中粘贴真实密钥、截图或个人金融数据。
