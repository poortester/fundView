# AI 投研设计说明

Trading Agent 采用保守的研究员辅助模型。它的目标是帮助用户整理证据、风险和执行条件，而不是自动交易。

## 设计原则

- 每个结论都应尽量绑定到可见数据或证据。
- 优先展示结构化证据，而不是依赖自由发挥的说服性文本。
- 区分市场、行业、风险、资讯和执行记录。
- 展示不确定性、触发条件和观察项。
- 记录建议是否被执行，并在后续用真实净值验证效果。

## 本项目不做什么

- 不做自动下单。
- 不实现高频交易逻辑。
- 不生成无法验证的买卖指令。
- 不复制第三方 AI 交易项目的代码、提示词或素材。
- 不把大模型输出当作权威结论。

## 后续改进方向

- 增加下跌、反弹、震荡等场景分析。
- 增加组合重仓重叠度检测。
- 支持自然语言投研提问。
- 改进授权数据 API 的适配层。
- 增加完全本地模式，避免把个人数据发送给外部 LLM 或 OCR 服务。
