# 数据源状态

本文档用于说明当前项目接入的数据源、用途和风险边界。

## 当前使用的数据源

- 东方财富行情接口：主要指数快照和市场概览。
- 东方财富 / 天天基金网页资源：基金净值、估值、持仓穿透和基金公告。
- 东方财富基金相关正式资讯：经过过滤，尽量避免普通论坛评论混入基金快讯。
- Turso / libSQL：保存用户持仓、净值快照、研究报告和建议执行记录。
- 百度 OCR：在配置密钥后，可选用于持仓截图识别。
- 大模型服务：可选接入 OpenAI 兼容接口或 Anthropic 兼容接口。

## 重要限制

- 东方财富 / 天天基金相关能力基于公开网页资源，并非官方稳定 SDK。
- 数据可能延迟、缺失、限流、结构变化或不可用。
- 基金快讯会偏向正式公告、报告和资讯，因此部分基金可能没有快讯。
- 大模型输出只是辅助整理，可能出错，未配置模型时会回退到规则逻辑。
- OCR 可能会把个人持仓截图发送给第三方服务，请谨慎使用。

## 生产环境建议

- 将网页抓取式数据源替换为授权行情和基金数据 API。
- 对外暴露服务前增加登录认证。
- 增加请求限流、缓存和数据源错误降级。
- 为个人持仓、截图和模型请求建立明确的数据保留策略。
