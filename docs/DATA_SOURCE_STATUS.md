# 数据源与证据链状态

更新时间：2026-07-03

## 已接入

- 东方财富指数行情：沪深300、创业板指、上证指数、深证成指、恒生指数、恒生科技、纳斯达克100、标普500、美元/人民币、十年国债收益率。
- 东方财富基金净值：按持仓基金代码拉取最近约 180 个净值点，并取实时估值。
- 东方财富基金持仓穿透：重仓股、行业板块、概念、资产配置，持久化到 fund_exposures 表。
- Turso 持仓库：成本、份额、目标仓位、标签、手工编辑记录。
- 百度 OCR：持仓截图识别入口已接入。
- 分析报告与建议库：analysis_reports / advice_items 已启用，每日生成可追溯建议。
- LLM 解释层（可选）：配置 LLM_API_KEY 后，分析报告摘要自动附带 AI 解读。

## 已落地能力

- `/api/data/refresh`：刷新真实行情、基金净值、持仓穿透，并在刷新后自动生成当日分析报告与建议。
- `/api/analysis/refresh`：单独触发分析报告生成，写入 analysis_reports 与 advice_items；若配置 LLM key 则附加 AI 解读。
- `/api/advice/:id`：更新建议状态（已执行 / 跳过），用于复盘统计。
- `/api/review`：GET 读取复盘笔记，PUT 保存复盘笔记。
- `/api/review/stats`：返回历史建议执行率、跳过率、等级分布、近 14 天逐日统计、执行后收益归因汇总。
- `/api/review/attribution`：返回已执行建议的事后收益归因（基线净值、当前净值、胜率、平均收益）。
- `/api/data-sources/status`：返回市场行情、基金净值的数据源健康状态。
- `/api/funds/:id/evidence`：返回单基金证据链，包括净值来源、基准指数、仓位、近 30/90/180 点收益、最大回撤和波动。
- 交易日动态计算：按本地时间取最近工作日，不再硬编码，跨日不会错位。
- 组合板块暴露聚合：按持仓权重聚合各基金重仓行业，输出集中度告警（单一板块超 25%）。
- 前端市场数据卡展示真实源。
- 前端基金归因详情展示真实数据证据。
- 前端"今日建议"卡片：展示落库建议并支持执行/跳过状态流转。
- 前端"组合板块暴露"区块：进度条展示各行业权重与涨跌，集中度告警。
- 前端"复盘统计"区块：执行率、执行/跳过/待处理计数、近 7 天趋势条。

## 当前限制

- 港股恒生科技已接入真实源，但部分基金（QDII）的海外指数相关性仍需完善。
- LLM 解释层为可选增强，未配置 key 时降级为纯规则总结。
- 归因统计需要用户实际标记建议"已执行"后积累数据，系统自动记录基线净值。
