# 产品方向

Trading Agent 是一个本地优先的个人基金投研工作台。

## 目标

- 看清当前基金持仓。
- 跟踪估值、今日盈亏和累计盈亏。
- 解释市场、行业和持仓暴露。
- 展示正式基金公告、基金资讯和数据源状态。
- 记录研究建议，并在后续验证建议效果。
- 支持可选的大模型辅助总结，但不强制依赖大模型。

## 非目标

- 不做自动交易。
- 不接入券商交易。
- 不作为公开多用户 SaaS。
- 不承诺投资收益或投资建议准确性。
- 不替代授权金融数据终端。

## 近期路线图

1. 完成本地启动、开源说明和密钥保护。
2. 改进数据源适配和失败状态展示。
3. 增强研究员证据卡片和结论可追溯性。
4. 增加 LLM / OCR 的隐私开关和发送前确认。
5. 增加本地持仓导入导出，并避免泄露密钥。

## 风险姿态

这个应用应被视为个人决策辅助工具。产品表达应优先透明、可审计和保守，而不是给出激进确定的交易建议。
